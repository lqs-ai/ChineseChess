## java联网中国象棋

> 简介：这是大二课设做的基于javase综合的的一个项目，用到的技术栈有Swing，算法，反射，文件，IO，多线程，Socket，栈等数据结构，实现了走棋，保存棋局，读取棋局，悔棋，求和，认输，保存玩家信息，听音乐，切换棋子，背景图，棋盘等功能。
